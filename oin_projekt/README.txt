
oin_projekt is a subtheme of default bootstrap theme for drupal 8. The
subtheme was developed as part of the project Objekte im Netz (aka OiN).
Bootstrap, version 8.x-3.20 provides a template directory for subthemes.

base-theme: bootstrap, version 8.x-3.20

- footer.css: customized footer based on footer of so called oin-fachsysteme
  (oin_musik).
- header-bootstrap.css: no adaptions
- adaptions.css: all adaptions away from bootstrap default theme

INSTALLATION:
- install bootstrap from drupal site
- clone oin_projekt from git 
- clear cache in your drupal system
- install and actived theme at menu item appearance
